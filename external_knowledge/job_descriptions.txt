JOB DESCRIPTION 1
Title: Junior AI Engineer
Location: Venice, Italy
Company Type: AI Solutions Startup

About the Role:
We are looking for a Junior AI Engineer to support the design and development of AI-powered applications. The candidate will work on machine learning pipelines, prompt-based systems, and basic LLM application development. The role includes working with Python, APIs, model evaluation, and integration of AI modules into user-facing applications.

Responsibilities:
- Develop and test AI prototypes using Python
- Build simple applications using LLM APIs
- Preprocess and analyze structured and unstructured data
- Assist in machine learning experimentation and model evaluation
- Collaborate with product and technical teams to improve AI features
- Document technical workflows and design decisions

Requirements:
- Degree in Artificial Intelligence, Computer Science, Data Science, or related field
- Good knowledge of Python
- Familiarity with machine learning libraries such as scikit-learn, TensorFlow, or PyTorch
- Basic understanding of NLP, computer vision, or data analysis
- Ability to work with APIs and integrate AI into applications
- Good communication skills in English
- Italian language is a plus

Preferred:
- Experience with Streamlit or similar rapid prototyping tools
- Familiarity with LLM workflows or prompt engineering
- Exposure to cloud tools such as Microsoft Azure
